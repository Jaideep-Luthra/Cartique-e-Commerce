# 🛒 Cartique – MERN E-Commerce Website with Voice Recognition

Cartique is a modern **full-stack e-commerce web application** built using the **MERN Stack (MongoDB, Express.js, React.js, Node.js)**.

The platform enhances traditional online shopping by integrating **voice command functionality**, allowing users to browse products, search, and manage their cart using simple voice inputs.

---

## 🚀 Project Overview

Cartique is designed to deliver:

- 🛍 Seamless product browsing  
- 🛒 Efficient cart management  
- 🔐 Secure authentication system  
- 🎙 Voice-based navigation & search  
- 📦 Smooth checkout experience  
- 📊 Admin dashboard for full control  

The application provides both traditional UI interaction and hands-free voice commands to improve accessibility and user experience.

# 🎙 Voice Command Integration

Cartique integrates the **Web Speech API** to enable:

- 🔍 Voice-based product search  
- 🛒 Add products to cart via voice  
- 🧭 Navigate pages using commands  
- 🔊 Audio feedback confirmation  

This feature enhances:
- Accessibility
- Hands-free shopping
- Modern interactive UX
